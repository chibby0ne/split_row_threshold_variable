Detailed Description

Parse the configuration XML file to configure the module instances.

This class parsing a XML configuration file and search for the "module" sections. It provides a function the configure a module with the parameter given in the XML file. The assignment of the XML section is done over the instance name! In addition the module automatic instaziate the given parameter unit modules and propagate their ports into to global variable section. After that the global variables were propagated into the module sections.
