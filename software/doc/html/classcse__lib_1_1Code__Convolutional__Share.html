Detailed Description

Share class for convolutional based codes.

This class contains functions to handle convolutional codes, based on a shift register.

enum cse_lib::Code_Convolutional_Share::TRELLIS_TERMINATION_ENUM

Enum type for the termination.

Enumerator:
TRUNCATION 

no additional termination is done, end state is random

TAILING 

the encoder end state is the zero state, additional bits were generated.

TAIL_BITING 

the encoder's start end state are the same, no additional bits were needed.

Encoder the given data with the given initial state and returns the end state.

The function uses the table and parameter from the Calculate_Trellis_Parameter() and the Calculates_State_Tables() functions

Parameters:
[in] input_data bits to encode
[out] output_data contains first all bits from the input_data and then the parity bits the parity bits from each time step are in a consecutive order.
[in] start_state State number of the encoders starting state.
[out] end_state State of the encoder after the encoding process
calculate_parity_bits = true Calculates the parity bits, = false does not calculate the parity bits. Can be used to determine the encoder's end state for a given input sequence (circular encoding).
