Detailed Description

template<class T>
class cse_lib::Status_Out_Plugin_Default< T >

Default plug-in for the Status_Out port (memory behaviour).

This plug-in is used by Status_Out if no other plug-in is selected. The default plug-in works as a simple memory for one value.

Each Write() call overwrites the current value. The Read() function returns the currently stored value.
