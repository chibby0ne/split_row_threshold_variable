Detailed Description

This class hold one configuration set for the modules.

You can copy this class, without copy the configuration sets, this allows for an memory efficient storage of the currently used configuration. This class does not manage the allocation and deletion of the objects behind the stored pointer. The memory management must be done outside. This implementation allows to copy the class without unneeded copy operations and memory efficient storage.
