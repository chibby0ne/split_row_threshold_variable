Detailed Description

template<class T>
class Buffer< T, 2 >

Generic data container with a dimensions of two (matrix).

Buffer class to store data into a matrix array. Class is a specialization of the buffer<T,dim> class with dim = 2. In addition if the ITPP_SUPPORT definition is set the class member functions support the Mat<T> data type of it++ library.
