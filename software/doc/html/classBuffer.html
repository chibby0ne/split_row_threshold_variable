Detailed Description

template<class T, int DIM>
class Buffer< T, DIM >

Buffer class with type and optional dimension as template parameter.

Buffer class stores data in a multidimensional array. The size of the buffer can be adjusted during runtime. Single values or all data of the buffer can be read or written. You can directly use the buffer at the beginning or/and end of your operations to read/write your values without copying them before or after the operation.

Template Parameters:
<class T> Data type of data which can be stored in class.
<int dim> Number of dimension of the buffer
