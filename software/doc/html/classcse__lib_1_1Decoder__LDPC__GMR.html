Detailed Description

LDPC decoder for the GMR standard (N = 950 ... 11136, P = 16 ... 87, R = 0.5 ... 0.9).

The LDPC decoder complies to

  • GMR Release 2, ETSI TS 101 376-5-3 V2.3.1 (2008-07) (GMPRS-1 05.003)
  • GMR Release 3, H36978 GMR-3G 45.003 V3.1.1 (2007-11) (GMR-3G 45.003)

GMR Release 1 (GMR-1) does not include LDPC codes.

De-shortening, de-puncturing and de-repeating is not included in the module.
