Detailed Description

template<class T>
class cse_lib::Interleaving_GMR< T >

GMR standard, (de-)interleaving functions.

This entity performs block interleaving and block deinterleaving. The interleavers comply to

  • GMR Release 2, ETSI TS 101 376-5-3 V2.3.1 (2008-07) (GMPRS-1 05.003)
  • GMR Release 3, H36978 GMR-3G 45.003 V3.1.1 (2007-11) (GMR-3G 45.003)

GMR Release 1 (GMR-1) does not include these interleavers.

Shortening, (de-)puncturing and (de-)repeating is included in the module (set xs, xp, and xr to values != 0). By default, the (de-)interleaver is used without these features.
