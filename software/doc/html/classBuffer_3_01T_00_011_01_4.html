Detailed Description

template<class T>
class Buffer< T, 1 >

Buffer class with one dimension (array).

Buffer class to store data into a matrix array. Class is a specialization of the buffer<T,dim> class with dim = 1. In addition if the ITPP_SUPPORT definition is set the class member functions support the Vec<T> data type of the it++ library. Class as an explicit it++ support, that mean to operate with the Vec<T> type from it++ instantiate the class with the type T and NOT with the type Vec<T> ! In addition it has many functions to read and write data to native C++ arrays.
