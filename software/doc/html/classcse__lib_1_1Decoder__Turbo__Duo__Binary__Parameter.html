enum cse_lib::Decoder_Turbo_Duo_Binary_Parameter::BCJR_STRUCTURE_ENUM

Enum for the TRELLIS structure which is used for decoding.

Enumerator:
NO_WINDOWING 

Normal forward backward recursion over the complete trellis

WINDOWING_ONE_TRELLIS 

Trellis is split into several equal sized parts, defined with the parameter window_length. The forward recursion is not windowed. The backward recursion uses a windowed scheme.

WINDOWING_TWO_TRELLIS 

Trellis is split into two independent parts. The first part uses a non windowed forward recursion and a windowed backward recursion. The second part uses a windowed forward recursion and a non windowed backward recursion. The values at the borders where exchanged between the two parts.
