Param<unsigned int> cse_lib::Interleaving_GMR_Parameter::bw_shortened

Bit width for the output value for de-shortened bits of the codeword.

This parameter is only used when mode = DEINTERLEAVE and xs != 0. In that case it is necessary to de-shorten the input data, i.e. set some output value to the maximum value that can be represented (for fixpoint values), which corresponds to the zero, that was encoded on the transmitter side. The module inserts 2^(bw_shortened - 1) - 1 into the codeword for these bits (float and int case!).

Param<unsigned int> cse_lib::Interleaving_GMR_Parameter::num_info_bits

Number of information bits of the LDPC codeword ( ).

Used to determine the column order during deinterleaving. The parameter is only relevant for num_bits_per_symbol = 4.
