Detailed Description

Parameter Class for the binary source module.

In this class all parameters for the binary source are defined. This class should not be instantiated itself, it should be used for derivation of a module.

Member Data Documentation

Param<unsigned int> cse_lib::Source_Bits_Parameter::crc_poly

CRC code to add to the sequence. Only active if unequal to zero.

The last bits of the codeword are replaced by the CRC code of the previous bit sequence. The given polynomial determines the number of bits to replace. The length of the output sequence is not affected by the CRC feature. The LSB corresponds to the input flip-flop of the shift register.
