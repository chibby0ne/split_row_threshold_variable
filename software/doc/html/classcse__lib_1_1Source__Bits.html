Detailed Description

Binary source class.

This class generates random bit sequences.
In addition, the two modes ALL_ZERO and ALL_ONE are supported to get an all zero or all one sequence to perform, e.g., zero code word simulations. In addition, a CRC code can be included in the sequence if the parameter crc_poly is set.
