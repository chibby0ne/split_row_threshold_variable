class cse_lib::Status_Out< T, DIM, F >

Output status port class of the modules.

Template Parameters:
T determines the data type of the status port.
F gives the plug-in type class (e.g, calculate the mean value of all values which will be written. The standard plug-in is the simple data storage.
dim determines the dimension of the port
