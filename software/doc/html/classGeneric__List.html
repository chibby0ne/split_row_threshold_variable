Detailed Description

template<class T, template< class T, typename > class Cont>
class Generic_List< T, Cont >

Template class to manage objects.

Template Parameters:
Cont can be any container class from the STL library, which can store objects of type T (e.g., vector ord dequeue).
T can be any type which can be stored in the STL list types

This class as an internal list to manage the inserted objects. A copy of the given object is stored.
