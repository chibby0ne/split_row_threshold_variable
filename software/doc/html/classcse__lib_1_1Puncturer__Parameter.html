Member Data Documentation

Param<Buffer<int> > cse_lib::Puncturer_Parameter::puncturer_table

Puncturer table to use.

The table definition is as follows: Whenever a negative value is found within the table, the value with the same index of the input value as the negative value of the table will be removed. For instance, the pattern 0, -1, 1, -1, 2, -1, 3, -1, ... will remove every second bit from the input stream. For depuncturing or padding, the sequence above will insert values as defined by the depuncturing_value parameter.
