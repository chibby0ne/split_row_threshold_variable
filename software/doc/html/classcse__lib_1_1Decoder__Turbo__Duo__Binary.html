Detailed Description

template<typename T>
class cse_lib::Decoder_Turbo_Duo_Binary< T >

Flexible Duo binary MAP turbo decoder, based on the BCJR algorithm.

This is a very flexible model, with different decoding algorithm. In addition it can act as floating or fix-point bit true model. The decoder can only process circular encoded codes.

Template Parameters:
T data type of the decoder (int = fix-point, float = floating point).
