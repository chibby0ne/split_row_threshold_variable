Detailed Description

template<typename T>
class cse_lib::Interleaver< T >

General table-based (de-)interleaver.

This module interleaves or deinterleaves data with the given interleaver table. Interleaver means that the interleaver reads the values sequentially and writes them to the position given by the table.
