Detailed Description

Namespace for the Creonic simulation environment (CSE).

This namespace contains all modules and their parent classes of the CSE. The standard namespace and the namespace of the helper functions is included in this namespace!

Enumeration Type Documentation

enum cse_lib::INTERNAL_STATE_RUN_ENUM

Enum type to control the internal state reset behavior of the modules.

Enumerator:
RESET 

Reset all internal states, default behavior

KEEP 

Keeps the internal states untoched. That measn the module begins with the state wich was set from the last call of the Run function

enum cse_lib::RETURN_VALUE_RUN_ENUM

Enum type for the return values of the Run functions.

This type is used as return type of the Run function of the modules. The module indicates whether and which stopping event occurs (e.g., iteration control or all simulation finished).

Enumerator:
NO_STOPPING_EVENT 

No stopping event occurs, normal case

STOP_CURRENT_SIMULATION_POINT 

Stop the current simulation point (e.g., all blocks have been simulated)

STOP_CURRENT_BLOCK 

Stop the current processed block (e.g., iteration control has detected a valid code word)

STOP_COMPLETE_SIMULATION 

Stop the complete simulation (e.g., further simulation point does not make sense)
