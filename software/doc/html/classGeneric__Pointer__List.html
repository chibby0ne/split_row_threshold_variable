Detailed Description

template<class T, template< class T, typename > class Cont>
class Generic_Pointer_List< T, Cont >

Template class to manage objects (list of object pointers).

Template Parameters:
Cont can be any class (container) from the STL library, which can store objects pointers of type T.
T can be any type which can be stored in the STL list types

This class as an internal list to manage the inserted objects. Only the object pointers are stored. That means no copy will be created, but the list does not care about stored objects which were deleted! If you delete an inserted object outside you have to removed it out of the list before.
