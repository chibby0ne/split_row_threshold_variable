Detailed Description

Functions that can be shared between multiple mappers.

In particular this module contains the mappings for many modulation schemes in form of look-up-tables (LUTs). The LUTs are of such a form, that the index address corresponds to the bit sequence of the contained constellation point at that address. The MSB of the LUT address corresponds to the bit that is being transmitted first!
