Generic function to generate a puncturer table.

This function generates a puncturer table as required, e.g. for turbo decoding. The function uses two input buffers that contain the lengths and the patterns for puncturing for each single codeword part (e.g., systematic or different parity information). The function can be considered to be working on bits. The pattern is expected to be in the format of zeros and ones, where a one means, transmit this bit, and zero means to puncture the bit. The puncturer table contains negative values for bits to puncture and positve index values for bits to keep.

Parameters:
lengths Gives the lengths of the single codeword parts.
patterns Gives the puncturing pattern for the different codeword parts.
puncturer_table Output of the function.

Example 1: Turbo Code, Rate 1/3, Tail-Biting, 100 information bits
lengths = "100, 100, 100";
patterns[0] = "1";
patterns[1] = "1";
patterns[2] = "1";

Example 2: Turbo Code, Rate 1/3, Tailing, 8-states, 100 information bits
lengths = "103, 103, 103";
patterns[0] = "1";
patterns[1] = "1";
patterns[2] = "1";

Example 3: Duo-Binary Turbo Code, Rate 2/5, Tail-biting
lengths = "100, 100, 100";
patterns[0] = "1";
patterns[1] = "1";
patterns[2] = "1, 1, 0, 0"; // Bit-wise pattern

Example 4: Duo-Binary Turbo Code, Rate 3/4, Tail-biting
lengths = "100, 100, 100";
patterns[0] = "1";
patterns[1] = "1, 1, 0, 0, 0, 0"; // Bit-wise pattern
patterns[2] = "0, 0, 0, 0, 0, 0"; // Bit-wise pattern, can be reduced to "0"

Returns:
the number of punctured positions
